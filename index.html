Site vitrine – Haut En Couleur (Laravel)

Ce projet a été réalisé dans le cadre d’un stage de 6 semaines chez l’entreprise Haut En Couleur, spécialisée dans la rénovation intérieure et extérieure. L’objectif était de créer un site vitrine moderne et responsive permettant de présenter l’activité de l’entreprise et de faciliter la prise de contact avec les clients.

J’ai conçu le site en utilisant le framework Laravel, avec une architecture MVC complète, un système d’authentification sécurisé (Laravel Breeze), une interface admin, un formulaire de contact fonctionnel avec envoi d’e-mails, et une gestion dynamique des contenus (services, réalisations...).

  • Framework : Laravel
  • Frontend : Blade, Bootstrap, JavaScript
  • Backend : PHP + MySQL
  • Fonctionnalités : formulaire de contact, interface admin, pages dynamiques (services, réalisations)
  • Outils : Figma (maquettes), Notion (organisation), GitHub, MailTrap (test e-mails)

Application GNotes – Gestion de notes BTS

GNotes est une application JavaFX développée dans le cadre de ma 2e année de BTS SIO SLAM, dont l'objectif est de permettre aux enseignants de gérer les notes, les appréciations et les statistiques des étudiants de manière fluide et hors ligne.

Chaque professeur peut créer ses propres évaluations, entrer les notes, ajouter des appréciations, et visualiser en temps réel la moyenne de la classe, ainsi que les notes minimales et maximales, mises en évidence par un système de couleurs.

Les données sont stockées localement en JSON pour un usage hors ligne, et peuvent être synchronisées vers une base MySQL distante via une API REST. Le projet applique la méthodologie SCRUM, les principes du développement piloté par les tests (TDD) et intègre une gestion rigoureuse des exceptions.

  • Langages : Java, JavaFX
  • Stockage local : JSON
  • Backend/API : Spring Boot
  • Base de données distante : MySQL
  • Fonctionnalités : statistiques, couleurs dynamiques, mode hors-ligne, vue des majors

GSB Étudiants – Application Laravel

Ce projet a été réalisé avec le framework Laravel et consistait à développer une application de gestion des visiteurs médicaux pour le laboratoire GSB. L’objectif était de permettre aux utilisateurs de visualiser, créer et gérer des rapports de visite pour chaque praticien rencontré.

L’application intègre une gestion des profils utilisateurs (visiteurs, délégués, administrateurs), des visites, des échantillons distribués, ainsi qu’un système d’authentification sécurisé. Une interface moderne et responsive a été conçue pour une utilisation fluide sur tout type d'appareil.

  • Framework : Laravel 10
  • Base de données : MySQL
  • Authentification : Jetstream / Sanctum
  • Fonctionnalités : rapports, visiteurs, praticiens, médicaments
  • Interface : Blade + Tailwind CSS
